Set

介绍

Set 是 ES6 提供的一种新的数据结构， 和数组一样，都可以用来存储有序的数据集合，但是它本身却不具备根据索引的访问某一个具体元素的能力，你可以按照插入的顺序迭代它的元素。 不过需要注意的是Set中的元素只会出现一次，即 Set 中的元素是唯一的。说了这么多，让我们来看一下Set的语法吧~
